All validated fields have to have standard getters

Import:

Add it in your root build.gradle at the end of repositories:

    allprojects {
        repositories {
          ...
          maven { url 'https://jitpack.io' }
        }
      }
      
Add the dependency

      dependencies {
                ...
                compile 'com.github.gregor9494:AndroidValidator:1.0.4'
        }
